# Realt Slack bot

Бот предназначен для поиска выгодного предложения по сдаче квартиры в аренду

## Установка

```javascript
git clone https://github.com/akwuh/realtbot/
cd realtbot/
npm i
```

## Запуск

```javascript
cd realtbot/
SLACK_TOKEN=<your-bots-slack-token-here> npm run start:production
```
